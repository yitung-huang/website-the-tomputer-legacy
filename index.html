I'm currently re-making my sister's website - a medical cat blog which she has designed and created using Wix (yes, she's a cat lady).

Why are you re-making a website that is fully functional?

I'm glad you've asked. There are a few different reasons. I want to:

  • Make the website more responsive - the Wix editor seems to only allow creating the website in desktop and mobile layouts only. This is not ideal as some people use tablets or smaller laptops, but websites made by Wix don't look great on these devices.
  • Improve user experience - there are some minor improvements that could be made such as changing the colour of links and buttons when the mouse hovers over them.
  • Make the website easily maintainable - it's a bit stressful trying to update the website on Wix because you never know if you're accidentally clicking and dragging an element. You could be trying to add something to your website and the next thing you know something's been accidentally deleted or moved and everything is out of place!
